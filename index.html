Research Projects

Generative AI for Training Dataset Augmentation

This project explored a data augmentation pipeline using StyleGAN to generate synthetic Cahn-Hilliard patterns. This approach circumvented the need for extensive Finite Element simulations, which are typically time-consuming. With only 1000 real samples, the StyleGAN produced additional data, enabling the training of a robust deep learning model for precise strain energy prediction in materials. This hybrid training strategy significantly improved the model's accuracy, demonstrating the value of synthetic data in limited-sample scenarios.

CNNs for Material Mechanical Behavior Prediction

In this work, we delved into CNN architectures to directly predict full-field mechanical quantities across heterogeneous materials. Our exploration encompassed several U-Net variations, and through empirical analysis of multiple mechanical datasets, we introduced Multires-WNet. This U-Net adaptation has shown promising results in case studies predicting strain fields, stress fields, damage fields, and crack path trajectories.

SarcGraph: A Python Package for Cardiomyocyte Analysis

We developed SarcGraph, a Python package specifically designed for the detection and tracking of Z-discs and sarcomeres in images and videos of beating stem-cell-derived cardiomyocytes. Our approach features a new graph-based algorithm for faster and more accurate sarcomere detection, coupled with a tracking system enhanced by DBSCAN clustering. Crafted to be modular and user-friendly, the package accommodates researchers of varying coding expertise. Users can readily adapt and extend the code, or directly apply it to their datasets.
